# 🏡 Boston Housing Price Prediction

## Overview

This project was my first introduction to machine learning using Python and Scikit-Learn.

The objective was to explore the Boston Housing dataset, understand which housing characteristics influence property prices, and build a Linear Regression model capable of predicting house values.

Rather than focusing only on the final prediction, this project helped me understand the complete machine learning workflow—from preparing the data to evaluating the model's performance.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn

---

## Project Workflow

- Imported and explored the dataset
- Cleaned the data by removing unnecessary columns
- Split the data into training and testing sets
- Trained a Linear Regression model
- Predicted house prices
- Compared predicted values with actual values
- Calculated residuals
- Evaluated the model using:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score
